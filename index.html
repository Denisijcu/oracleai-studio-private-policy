Política de Privacidad de OracleAI

Última actualización: 18 de abril de 2026

En Vertex Coders LLC, la privacidad de nuestros usuarios es nuestra prioridad fundamental. Esta Política de Privacidad describe cómo OracleAI recopila, utiliza y protege su información.

1. Recopilación de Información

OracleAI es una plataforma de Inteligencia Artificial multimodal. Recopilamos los siguientes datos para funcionar:

  • Datos Visuales: Acceso a la cámara para análisis de objetos, detección de caídas y asistencia visual.
  • Datos de Audio: Acceso al micrófono para comandos de voz y traducción en tiempo real.
  • Información del Dispositivo: Modelo de hardware y versión del sistema operativo para optimizar el rendimiento de la IA local.

2. Procesamiento de Datos (Sovereign AI)

A diferencia de otras aplicaciones, OracleAI prioriza el procesamiento local (Edge Computing). Gran parte del análisis visual y lógico se realiza directamente en su dispositivo para minimizar el envío de datos a servidores externos.

3. Uso de Google AI (Gemma 4)

Para funciones avanzadas de razonamiento, OracleAI utiliza la API de Google AI (Gemma 4). Los datos enviados se limitan estrictamente a lo necesario para generar la respuesta solicitada y están sujetos a las políticas de seguridad de Google Cloud.

4. Seguridad de los Datos

Implementamos medidas de seguridad de nivel industrial para proteger sus datos contra el acceso no autorizado, alteración o divulgación. No vendemos ni compartimos su información personal con terceros para fines comerciales.

5. Permisos de Android

La aplicación solicitará permisos explícitos para:

  • Cámara: Necesaria para los modos Sentinel y Educación.
  • Micrófono: Necesario para el modo Conversación.
  • Almacenamiento: Para guardar registros de análisis locales si el usuario lo desea.
